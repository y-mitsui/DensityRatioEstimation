Density Ratio Estimation
=============

Python liblary of the Density Ratio Estimation (Sugiyama).

## Features
* Direct estimate two ratio of probability distribution as high accuracy as follows.
 
```
r(sample) = P_x(sample) / P_y(sample)
```

## Usage
Read lsif.py.

```
python lsif.py
```
